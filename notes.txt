--- Section 1 - Loopback Big Picture
Highly extensible open-source Node.js framework to build dynamic end-to-end REST APIs.

Why Loopback? 
    Selain loopback, kita perlu include semua package : database, integrate API Documentation, Login Module, Storage Component.
    Loopback provides all these features.
        Bisa connect ke sql atau mongo
        Default API Explorer 
        Create Dynamic Rest APIs
        Separable components fileStorage, third party, Oauth2.0 udah termasuk
        Punya access control untuk api yang kompleks
